# Controls

### Menus
| Command          | Key        |
|------------------|------------|
| Navigate         | Arrow Keys |
| Submit           | Enter      |


### Movement/Attacking Directions

This game uses traditional roguelike controls for movement.

| | | |
|-|-|-|
| y | k | u |
| h | @ | l |
| b | j | n |

### Commands
| Command          | Key   |
|------------------|-------|
| Inventory        | i     |
| Interact/Pick Up | .     |
| Equip            | e     |
| Drop             | d     |
